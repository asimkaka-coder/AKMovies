# MVVM Movies App

Learning pagination by building a tv shows app
## Features

- Display Popular Shows
- Details about show


## Libraries and Dependencies

- Dagger Hilt
- Retrofit
- Paging 3
- Flow Live Data
- Coroutines
- Material Design
- View Models
- Recycler View
- Coil
- Jetpack Navigation
